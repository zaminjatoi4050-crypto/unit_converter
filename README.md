# 🔄 Unit Converter Pro — Python GUI
### Ultra Pro Max Edition | by Jatoi

## ▶️ How to Run

### Step 1 — Install library
```
pip install customtkinter
```

### Step 2 — Run the app
```
python unit_converter_gui.py
```

## ✨ Features
- 📏 Length — 8 units
- ⚖️ Weight — 6 units
- 🌡️ Temperature — 3 units (°C, °F, K)
- 🚀 Speed — 5 units
- 🔢 Formula display
- ⇄ Swap units
- 📜 Conversion history
- ⌨️ Press Enter to convert
- 🌙 Dark Pro UI
